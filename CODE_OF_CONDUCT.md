# Code of Conduct

Droidsmith adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
Short version, in our own words:

1. **Be kind.** Assume good faith. Disagreement is fine; contempt isn't.
2. **Focus on the work.** Talk about code, design, evidence. Don't talk
   about people.
3. **Keep politics off the issue tracker.** Threads that aren't about
   Droidsmith get closed, no debate.
4. **Respect the maintainers' time.** A short, clear bug report with a
   pasted heartbeat block is worth ten venting paragraphs.

If you experience or witness conduct that violates this code, email the
project maintainers (see [SECURITY.md](SECURITY.md) for the contact
address). Reports stay confidential.

Sanctions, in escalating order:

- A private warning explaining the issue.
- A public note on the offending thread.
- A timed comment ban on the repo.
- A permanent ban.

Maintainers commit to applying these consistently and to documenting
ban decisions in this repo's `docs/moderation-log.md` once that file
becomes necessary.
